このプロジェクトファイルはMicrosoft Visual Studiio上で動作するC#で書かれたGNSS関係のクラス群です。
現時点ではフォームアプリ開発環境しか考慮していません。
Visual C++, Visual Basic, Visual C#などのVisual Studio開発環境で利用可能です。 
参考URL：http://okachibi.web.fc2.com/dotnet/call_csdll_01.html

本クラス群は自分が必要な分だけ作りためたものです。
測地系はWGS84とGRS80しか考慮していません。

当面は、Blh座標系もしくはECEF座標系からENUローカル座標系へ変換するメソッドを実装することを目指します。
それが終わったら、衛星位置を処理する構造体・クラス群を整備する予定です。
